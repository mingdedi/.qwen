# SKILL.md 常见设计模式

## 模式 1: 工具封装器 (Tool Wrapper)

让 AI Agent 秒变某个库/框架的专家。

```yaml
---
name: api-expert
description: FastAPI development best practices. Use when building, reviewing, or debugging FastAPI applications.
---
```

## 模式 2: 生成器 (Generator)

从可复用模板生成结构化文档。

```yaml
---
name: report-generator
description: Generate structured reports from templates. Use when user asks to create a report, summary, or document.
---
```

## 模式 3: 规范加载器 (Convention Loader)

动态加载内部文档并作为"绝对真理"执行。

```yaml
---
name: coding-standards
description: Apply team coding standards. Use when reviewing or writing code in this project.
---
```
