# SKILL.md 发布前检查清单

在发布 SKILL.md 前，确认以下事项：

- [ ] YAML 前置元数据格式正确（`---` 包裹）
- [ ] `name` 字段仅使用小写字母、数字和连字符
- [ ] `description` 字段清晰描述触发场景和关键词
- [ ] 正文包含 "When to use this skill" 章节
- [ ] 工作流步骤编号清晰、可执行
- [ ] 工具调用规范完整（参数、返回值、错误处理）
- [ ] 提供了输入输出示例
- [ ] 说明了边界案例处理方式
- [ ] 引用了必要的资源文件
- [ ] 主文件控制在 500 行以内
- [ ] 复杂内容已拆分到 `references/` 目录
